Simple slider framework for simple projects. 8 KB

Settings Mouse Controll Items count Slide Group Margin Right Set slider loop Set autoplay Autoplay delay Slider navigation
Table with properties and their description
Properties Values Description
mouseControll type Boolean (true/false) Allows you to scroll through the slider with the mouse.
The default value is true
items type: Number You can specify the number of items for the visible area.
The default value is 1
slideGroup type Boolean (true/false) This allows you to flip through the slider group. Each group gets the number of slides that you specify in the items property.
The default value is false
marginRight type: Number You can specify the distance between items (in pixels).
The default value is 0
loop type Boolean (true/false) This property allows you to return to the first and last slide.
The default value is false
autoplay type Boolean (true/false) This property allows you to twist the slider automatically.
The default value is 3000 (milliseconds = 3 seconds)
delay type: Number This property allows you to set the speed of automatic scrolling.
The default value is 3000 (milliseconds = 3 seconds)
navigation type: Object Contains two properties.
First:
prev: ".className of element that calls scroll to the left"

Second:
next: ".className of element that calls scroll to the right"
